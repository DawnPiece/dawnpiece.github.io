# Abilities Guide

The server contains an entire ability system that was created in the DawnPiece Mod. Abilities include a Haki system, Fighting Style System, Race System and Devil Fruit System. There are also General Stats that are tied to each player and are separate from each system, but will assist you as you progress. Each ability system has its own leveling structure as well. Specific information about types of abilities and leveling system can be found on their respective pages below.
