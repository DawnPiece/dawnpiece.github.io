# Races

Races are based on the different races in One Piece. Start by choosing a Common Race in the Main Menu. Maxing out one race in each tier unlocks all races in the next tier to choose from. Higher-tier races are stronger at base value.

## Leveling

Each race levels independently so you can switch races and always retain your experience and skill levels for all races you already leveled up. Higher-tier races require more XP to level up. Unlocking all abilities requires reaching max race level.

All races level up by using the abilities. Experience is tracked by the damage dealt, damage protected, health restored, distance covered from dashes, etc. 

## Skill Points

Skill points can be assigned in the Main Menu. Each ability maxes out at Level 5 costing 10 skill points total. Maxing out all abilities requires reaching max race level.
