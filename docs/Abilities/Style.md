# Fighting Styles

Styles are based on the different non devil fruit related fighting styles in One Piece. Start by choosing a Style in the Main Menu. There are no style tiers, so each style levels independently up to Level 60.

## Leveling

Each style levels independently so you can switch styles and always retain your experience and skill levels for all styles you already leveled up. Higher-tier styles require more XP to level up. Unlocking all abilities requires reaching max style level.

All styles level up by using the abilities. Experience is tracked by the damage dealt, damage protected, health restored, distance covered from dashes, etc. 

## Skill Points

Skill points can be assigned in the Main Menu. Each ability maxes out at Level 5 costing 10 skill points total. Maxing out all abilities requires reaching max style level.
