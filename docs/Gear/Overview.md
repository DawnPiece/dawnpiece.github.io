# Gear Guide

The DawnPiece mod features a full custom gear system. Every NPC drops a unique set of armor including a helmet, chestplate, leggings, boots, and a weapon where applicable. Over time, every character will have a complete gear set available.

Use the links below to explore each section:

- ⚒️ [Blacksmiths](Blacksmith.md) - Information related to upgrading, repairing, salvaging, rolling, constructing, and runes.
- ⚔️ [Melee Weapons](MeleeWeapons.md) - Information related to weapon damage, stats, types, restrictions, upgrading, and repairing.
- 🛡️ [Armor](Armor.md) - Information related to armor resistance, stats, types, restrictions, upgrading, and repairing.
- ⚜️ [Set Bonus](SetBonus.md) - Information related to every armor set bonus.
